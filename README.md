# Rainfall - 42

## Overview

**Rainfall** is a cybersecurity project from school 42.
The goal is to explore **ELF-binary exploitation**, by connecting to and interacting with a VM to solve challenges.

The project focuses on:
- Exploiting insecure services
- Binary reverse engineering
- Applying practical pentesting techniques

Like a CTF, each level presents a challenge that must be solved to progress to the next.

---

## What you learn

- Binary reverse engineering
- Exploitation of network services
- Security auditing mindset

---

## Disclaimer

This project is for **educational purposes only**.  
All techniques must be used only in legal and controlled environments.
